IfM Digital Hospitals project
=============================

This Sphinx website describes a specification for a healthcare digital twin, using the Histopathology Department at Addenbrooke's Hospital, Cambridge as a case study. This site is part of the `Digital Hospitals <https://www.ifm.eng.cam.ac.uk/research/dial/research-projects/current-projects/distributed-information-and-automation-laboratory/>`_ project at the Institute for Manufacturing (IfM), University of Cambridge.
